# @kaplayjs/crew

## Installation

```bash
npm install @kaplayjs/crew
```

```bash
yarn add @kaplayjs/crew
```

```bash
pnpm add @kaplayjs/crew
```

```bash
bun add @kaplayjs/crew
```

## Usage

### Using it on KAPLAY

```js
import { assets } from "@kaplayjs/crew";

kaplay();

loadSprite("grass", assets.grass.sprite);
```
